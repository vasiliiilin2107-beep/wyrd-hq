## УСТАВ РАБОТЫ

### Закон 1 — Библиотекарь = единственная точка входа
Ни один отдел не обращается напрямую к читателям, Архивариусу или Писателям.
Все запросы — только через Библиотекаря.

### Закон 2 — Архивариус управляет фондом
Архивариус знает что есть и чего нет.
Читатели работают только по его заданию.
Без задачи — читатель спит. Спящий читатель не расход.

### Закон 3 — Писатель не принимает прямых заявок
Писатель пишет только то что передал Архивариус через Библиотекаря.
Он не отвечает на вопросы отделов. Он пишет в фонд.

### Закон 4 — Карантин непробиваем
Всё входящее (от Хугинов, Скрайба, любого источника) проходит Карантин.
Мусор, дубли, яд — не попадают в фонд.

### Закон 5 — Мусорщики работают постоянно
Фонд чистится регулярно. Дубли убираются. Устаревшее помечается.
Библиотека живёт — значит обновляется.

### Закон 6 — Профобразование учится, не мешает
Закуток Профобразования — тихое место.
Они берут знания из фонда, пишут промты, не отвлекают остальных.

## ДОЛЖНОСТНЫЕ ОБЯЗАННОСТИ

### Главный Библиотекарь
- Принимает заявки от всех отделов мира
- Передаёт запрос Архивариусу: "есть такое в фонде?"
- Получает ответ: есть → отдаёт отделу / нет → задача уходит в работу
- Принимает готовые материалы от Писателей → кладёт в фонд
- Ведёт учёт: кто что запросил, кто получил, что добавлено

### Архивариус
- Знает полный фонд библиотеки (индекс тем, категорий, дат)
- Отвечает Библиотекарю: есть / нет / частично
- Если нет → формирует задачу для читателей: тема, язык, приоритет
- Назначает задачи конкретным читателям (не всем сразу)
- Следит за качеством входящего от читателей
- Будит нужного читателя, усыпляет ненужного

### Писатель
- Получает от Архивариуса: тему, исходники, задачу
- Пишет: резюме темы, содержание, отсылки к смежным темам
- Не выдумывает — только обрабатывает то что есть в фонде
- Сдаёт Библиотекарю: готовый материал + ссылки на источники
- Один писатель = одна задача за раз

### Скрайб
- Принимает EN видео/аудио/текст
- Транскрибирует → переводит на русский
- Передаёт Почтальону → Карантин → Библиотекарь
- Работает автономно, докладывает о выполненном

### Мусорщик
- Регулярно сканирует фонд на дубли
- Находит устаревшее (по дате и актуальности)
- Помечает к удалению → докладывает Библиотекарю
- Удаляет только с подтверждения Библиотекаря

### Читатель (×19)
- Получает задачу от Архивариуса: что искать, язык, приоритет
- Ищет, собирает, передаёт Архивариусу
- Без задачи — режим сна (не тратит ресурс)
- Резервные читатели активируются при высоком спросе

### Профобразование
- Изучает что есть в фонде по своей теме (профессия конкретного бота)
- Пишет и обновляет промты на основе реальных людей этой профессии
- Передаёт готовые промты Томасу (через Библиотекаря)

## ЦЕПОЧКА РАБОТЫ

```
ВХОДЯЩИЙ ПОТОК (пополнение фонда):
Хугины / Скрайб → Почтальон → Карантин → Библиотекарь → Архивариус → Писатель → 📚 Фонд

ИСХОДЯЩИЙ ПОТОК (выдача знаний):
Отдел → заявка → Библиотекарь → Архивариус
                                      ↓
                    ЕСТЬ → Библиотекарь → Отдел
                    НЕТ  → Архивариус → читателю задача
                                      ↓
                           Читатель нашёл → Архивариус
                                      ↓
                           Писатель упаковал → Фонд
                                      ↓
                           Библиотекарь → Отдел

УБОРКА (постоянно):
Мусорщик → сканирует → докладывает Библиотекарю → удаляет
```
