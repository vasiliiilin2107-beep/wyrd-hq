### Что сделано

**Карта мира (7 слоёв) — `/hq` вкладка 🗺️**
- Заменили текстовые карточки веток на визуальную карту из 7 слоёв
- Слои: ЗАМЫСЕЛ → ШТАБ → ТОМАС → СТУДИЯ → БИБЛИОТЕКА → КАРАНТИН → ИНФРА
- Каждый слой показывает:
  - Статус ветки (● online / offline) из `/branches` API
  - Количество активных флагов из `/flags` API
  - Стрелку ▼/▲ состояния
- Клик на заголовок слоя — разворачивает/сворачивает
- Флаги отображаются чипами с иконкой по типу (⚓💡⚠️📡🔗📋📝)
- Клик по флагу — показывает/скрывает тело флага (описание)
- Кнопка `+ флаг` — добавить флаг прямо из карты (тип + название → POST /flags)
- Кнопка `×` на флаге — архивировать (PATCH status=done)
- Кнопка `↻` — обновить данные без перезагрузки
- ИНФРА-слой: всегда показывает статичные ноды (PostgreSQL/Redis/Qdrant/VPS/Coolify)
- Техническое: event delegation вместо inline onclick — надёжно
